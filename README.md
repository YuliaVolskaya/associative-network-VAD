# Моделирование аффективных характеристик лексики на основе ассоциативно-вербальной сети
*Итоговый проект по курсам "Компьютерная лингвистика", ВШЭ.*

Основная **задача** данного проекта – расширить разметку по аффективным параметрам для словника создаваемого ассоциативного словаря

**Материал**. База данных ассоциативного словаря русского языка объемом 156 тысяч пар стимул-ассоциация. 37 тысяч пар имеют разметку по трем аффективным параметрам *valence, arousal, dominance*. Эта разметка осуществлялась на основе базы данных аффективных норм (на основе массовых опросов среди носителей русского языка).

В основе исследования лежит **гипотеза**: слова, которые имеют схожие ассоциативные связи, обладают схожим смыслом и, как следствие, схожей эмоциональной окраской.

#### Подготовка данных
**Методология** подготовки данных на основе (Vankrunkelsven, 2018).


| Матрица              | Способ формирования                                                                 | Цель                                                                 |
|----------------------|-------------------------------------------------------------------------------------|----------------------------------------------------------------------|
| `freq_matrix`        | Формировалась на основе частот каждой леммы-реакции     | Зафиксировать сырые частоты ассоциативных связей                     |
| `ppmi_matrix`        | Positive Pointwise Mutual Information (PPMI), рассчитанная из `freq_matrix`        | Выделить статистически значимые ассоциации  |
| `similarity_matrix`  | Косинусное сходство между столбцами (леммами) транспонированной `ppmi_matrix`     | Получить дистрибутивное семантическое сходство слов-реакций          |

**Результат** – симметричная матрица размером 10711 на 10711.

#### Эксперименты и процесс обучения

**Таргет (y)**. Таблица размера 856 на 3, где 856 – это количество слов, для которых есть оценки VAD (словарь target_words). Колонки – три шкалы *valence, arousal, dominance*. 
**Признаки (X)**. similarity_matrix только для тех слов, которые есть в y. В качестве признаков выступают косинусные сходства этого слова со всеми остальными словами в ассоциативном корпусе. То есть признаками слова выступают его математические отношения ко всем остальным узлам языка.
В основе – **кросс-валидация на 5 фолдах**.

**Эксперимент 1. Сравнение базовых моделей**
1. Метод k-ближайших соседей (KNN):
*Признаки*. Матрица расстояний (вычислялась как дельта сходства: 1 - similarity_matrix).
*Модель*. KNeighborsRegressor с параметрами metric="precomputed" и взвешиванием по дистанции (weights='distance').

2. Гребневая регрессия (RidgeCV):
*Признаки*. Полная матрица косинусного сходства лемм.
*Модель*. Линейная регрессия с L2-регуляризацией (RidgeCV). Оптимизация параметра alpha по сетке np.logspace(-3, 4, 30) по критерию MSE.

3. Случайный лес (Random Forest):
*Признаки*. Полная матрица косинусного сходства лемм.
*Модель*. RandomForestRegressor с n_estimators=50, max_depth=10, max_features='sqrt'.

**Эксперимент 2. Стекинг**
*Базовые модели*. В качестве моделей первого уровня используются Out-of-Fold предсказания трех алгоритмов: KNN, Ridge-регрессия и Random Forest.
*Мета-модель*. Линейная регрессия с L2-регуляризацией (RidgeCV). Оптимизация параметра alpha по сетке np.logspace(-3, 4, 30) по критерию MSE.

**Эксперимент 3. RidgeCV + SVD**
Переход к латентно-семантическому анализу (LSA) для фильтрации шума и снижения размерности графа.
*Признаки*. Преобразование при помощи TruncatedSVD.
*Модель*: RidgeCV.

**Эксперимент 4. SVR + SVD**
Анализ нелинейных зависимостей в структуре ассоциативного графа.
*Признаки*. Сжатые векторы графа после TruncatedSVD.
*Модель*. Метод опорных векторов для регрессии (SVR) с радиально-базисным ядром (RBF).
*Гиперпараметры*: jптимизация параметров по сетке через GridSearchCV: C — [0.1, 1, 10, 50], epsilon — [0.01, 0.05, 0.1]. Автоматический масштаб гаммы (gamma='scale').

**Эксперимент 5. SVD + RuBERT**
Объединение двух уровней – структурной информации графа и контекстуальной семантики предобученной языковой модели.
*Признаки*. Графовые вектора (SVD) и текстовые эмбеддинги модели cointegrated/rubert-tiny2 (размерность 312). Объединенный вектор обрабатывался через StandardScaler.
*Архитектура*. Построена комбинированная схема предсказания в зависимости от природы шкалы:
Для шкал valence и dominance финальным предиктором выступил **RidgeCV** (alphas=np.logspace(-3, 4, 30)).
Для шкалы arousal применен нелинейный **SVR с RBF-ядром** и оптимизацией параметров C и epsilon через GridSearchCV.
**Размерность SVD** зафиксирована индивидуально для шкал: valence - 150, arousal - 100, dominance - 250.
